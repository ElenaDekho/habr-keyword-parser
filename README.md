# Парсер статей с Хабра

Скрипт парсит первую страницу свежих статей на [Habr](https://habr.com/ru/articles/), переходит в каждую статью и проверяет наличие заданных ключевых слов в:
- заголовке,
- основном тексте,
- тегах (хабах).

Если совпадение найдено — выводит дату, заголовок и ссылку на статью, а также сохраняет результат в файл `result.txt`.

## Ключевые слова

['дизайн', 'фото', 'web', 'python']

## Зависимости

requests==2.32.5
beautifulsoup4==4.14.2

## Установка

pip install -r requirements.txt

## Запуск

python main.py

Результат сохраняется в файл result.txt
